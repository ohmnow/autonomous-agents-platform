# Autonomous Agents Platform - Initial Release

We're excited to announce the initial release of the Autonomous Agents Platform!

<!-- truncate -->

## What's Included

### Core Features
- **Chat Interface** - Define your app through conversation with AI
- **Wizard Interface** - Step-by-step configuration for app specs
- **Autonomous Builds** - Launch builds that run in secure cloud sandboxes
- **Real-time Monitoring** - Watch your build progress with live activity feeds

### Packages
- `@repo/agent-core` - Core agent execution logic
- `@repo/database` - Prisma-based data layer
- `@repo/sandbox-providers` - E2B and Daytona integration
- `@repo/storage` - S3 and local storage providers
